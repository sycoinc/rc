Document Overview

Skewed Revival is one of the latest creations from Minecraft Website Templates designed for MinecraftMarket.

This theme has been made to be compatible with the premium and higher packages and works with the current version of the template editor (2.0).

Some of the themes features are as follows;

  • Pop Down Navigation that slides content to the back behind the menu items.
  • Custom locations for ALL Modules based on Module Title ( you can even add your own and modify the template to place them too! )
  • A Skewed feel to content, button and image sections to make it diffrent from every day block based themes.
  • Responsive Design, this site works on Mobile!

Basic Requirments:

  • Some HTML, CSS and Javascript knowledge is needed if you wish to customize this template further or change hard coded names for Module Sections
  • Premium or Enterprise MinecraftMarket Plan

Known Issues:

  • Sometimes the Modal Popups are slow to load on Mobile and Desktop. This seems to happen at random time and works fine other times. I believe it may be due to the load on the MinecraftMarket Servers or larger content in these sections.

Folder Structure

With Version 2.0 of the MinecraftMarket Template Editor there have been some changes to their system layout. Below is a overview of their new file struture along with where to place any custom files. Please click on the folders to see what files are inside.

Please note, files marked with an asterisk (*) are not used in this theme as are custom coded for layout customization.

  • modulesFiles related to all site content modules
    • goal.htmlGoals module (*)
    • text.htmlText box module (*)
    • recent_purchases.htmlRecent purchases module (*)
    • server_status.htmlServer status module (*)
    • customer.htmlTop customer module (*)
    • featured.htmlFeatured item module (*)
  • includesFiles related to all template include files such as footer & header
    • navbar.htmlControls the sites navigation
    • modal.htmlControls the modal popups
    • modules.htmlControls the modules container (*)
    • alerts.htmlControls the templates alert popups
    • header.htmlControls the header section of the website
    • footer.htmlControls the footer section of the website
  • staticThe location where users can add their own custom files (css / js)
    • cssCustom stylesheets are located here
      • custom.cssSkewed Revival custom stylesheet
    • jsCustom javascript files are located here
      • custom.jsSkewed Revival custom javascript
  • base.htmlThis file is the outer container for the whole website
  • options.htmlThis file is layout for the options presented to users for some packages
  • terms.htmlThis file is the layout for the terms & conditions page
  • category.htmlThis file is the layout for the category pages
  • item.htmlThis file is the layout for product items display
  • index.htmlThis file is the layout for the home text content
  • complete.htmlThis file is hte layout for the complete message after a successful purchase
  • maintenance.htmlThis file is the page that shows when a site is put into maintenance
  • checkout.htmlThis file is the layout of the checkout page

Template Installation

Currently MinecraftMarket has two ways to install a themplate. If you purchased your template via the MinecraftMarket Marketplace you can skip this step as it will be available for you to activate imediatly.

If you purchased this template on Minecraft Website Templates please follow the below steps.

Installing Skewed Revival Template Files

  • Extract Files from the downloaded Zip folder

    You will firstly need to extract all the files from the downloaded file you recieved when purchasing Skewed Revival from our website

  • Create Theme on MinecraftMarket

    Before you get started moving the downloaded template files over your market you will need to create a new template by following the steps below

    • Login and open your dashboard and navigate to Design > My Templates > Create Template
    • Name your tempalte and select the base template to be default
    • Click on confirm and after a few seconds your template should be created.
  • Open Files in HTML Editor

    After you have downloaded and extracted the required files for this template along with created your new template in the MinecraftMarket dashboard you will need to open all the downloaded files in a html text editor such as notepad++ or Atom.

  • Copy Downloaded File content to New template
    • Now go back to your dashboard and to the theme you created in step two and click on edit.
    • You will shortly be shown the templates file structure as shown in the folder structure section of this documentation. Here you will need to copy the files included in the downlaoded files you extracted in step 1 to their respective location.
    • To start lets move the code for base.html file into the template you created.
    • Open the base.html file from the downlaoded files inside your html text editor.
    • Open the base.html file inside the MinecraftMarket themplate you created in your dashboard.
    • Copy the content from inside the base.html file you donwloaded into the file you opened in the template dashboard. Make sure you replace all the content in the template dashboard file.
    • Click save on the template or press CTRL + S and it will save this file.
    • Repeat these steps for all files until you come to adding custom.css and custom.js
  • Adding Custom CSS and JS files
    • To add a new css or javascript file navigate to the static folder in your new template then click on the css or js folder.
    • To create a new custom file click on the green button and then name the file whatever you like but do not add the file extension (.css or .js)
    • Click on confirm to create your file. this may take a few minutes as static file creation is a little slow when creating, saving and opening.
    • Now you are ready to copy the content from the included files from your download into your template static files.
  • Enable Theme

    After you are happy head back to your templates by navigating to Design > My Templates and click on the enable text next to the required theme to enable this for your store.

  • Modify Market Information

    You are now one step closer to having a fully functional tempalate with a few more modifications and setting up categories, products,payment gateways and modules your theme will be looking along the lines of the demo.

Setting up the Custom Modules

With the Skewed Revival Template for MinecraftMarket it has been designed to show the built in modules in custom locations. You can even add more if you like (with a few modifications to the template files.). To add these modules into the correct locations they need to be named in relation to the code. You can modify these names yourself also if you do not like the naming conventions but for all besides the Featured 2 module they are the default module names.

  • Firstly we will be working with the Server Status & Text Box modules as the overview of this section. For the most part every modules custom location is set up according to this method.
  • All the modules are located in the base.html file and NOT the modules folder. Open the base.html file to get started.
  • Now scroll down the page until you see the following inline comment. 
    <!-- // Player Status Custom Module Block // -->
  • Under this inline comment you will find the code,

    Starting with. 

    {% for module in modules %}

    Ending with. 

    {% endfor %}

    This text is what holds all the modules for this specific section on the template. Without this piece of code the template will not show the modules at all and give you an internal error message on your store front.

  • Now there will be a lot of code inbetween these two tags but unless you are wanting to do some heavy editing you will only need to modify one section (or copy the inner section to make a new module / module position)
  • If you changed the names or made some custom names for your modules you will need to find the below code which sits inside the code snippets in step 4 of this section. The code snippet you will be looking for is below and is related to the Server Status. '...' indicates other code not shown in this overview. 
    
{% if module.title == 'Server Status' %}
...
{% endif %}
    To explain this code is rather simple, the code above looks for the module with the exact title which in this case is Server Status and then if this module is found and active it will be displayed in this location. please note module names are case sensitive so have to match the module name exactly
  • Next if you changed your Server Status module to be named something else change it here and save the file and see if it shows. You can even copy the entire code block (the above tags along with the content inside it) and make a second or even third server status module.
  • To add to this overview of this section the Skewed Revival theme support Two Featured Products to be displayed in the template. To find these two blocks search for the below inline comment. 
    <!-- // Start Custom Featured Item Module Block // -->
    Under this code you will see sections similar to the above overview but with the names Featured Item & Featured Item 2. Rename these to whatever you want your featured items modules to be named. Along with this feel free to change the code in the h4 tag as this controls the title that is of the module (this is also over many of the modules)

Modifying the Banner Image

The banner image is handled by the banner image upload option in the general settings of your marketplace.

Modifying the Social Icons

To modify the social icons or add additional icons navigate to your template once again and find the footer.html file which is located in the includes folder. Inside this file you will see links that relate to twitter, facebook and youtube. You can modify where these link to by editing the href tag or add more and use fontawsome icons which can be found at the fontawsome site.
